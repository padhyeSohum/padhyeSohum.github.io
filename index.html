What I'm Working On

Recently, I've developed an interest in reinforcement learning, a branch of machine learning. Reinforcement learning is an unsupervised technique which puts an agent, or model, a virtual environment, with the task of achieving the highest score possible by completing tasks. Reinforcement learning has many real-world applications, like robotics and game-development. Over the past few months, I've worked on creating reinforcement learning agents that could do various things, like play Super Mario Bros. and Rocket League, as well as maneuver a robot to sort trash.

Throughout this process, I learned about using Double Deep Q Networks and Proximal Policy Optimization, both reinforcement learning algorithms.
